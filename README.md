# tdviewer-py
`com.square_enix.android_googleplay.PTD`のデータを表示します。
画像と音声に対応予定。

必要なもの:
* ゲームのデータ(`/sdcard/Android/data/com.square_enix.android_googleplay.PTD/files/prim`)
* Python3

依存ライブラリ
* UnityPy
* sounddevice
* soundfile
* pysimplegui

## 依存ライブラリのインストール
pip install UnityPy sounddevice soundfile pysimplegui
